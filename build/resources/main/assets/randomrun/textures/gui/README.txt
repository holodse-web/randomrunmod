TEXTURES FOR RANDOMRUN MOD
==========================

Required textures:

1. logo.png (256x128 pixels)
   - Main logo for the mod menu
   - Should have transparent background
   - Purple/violet color scheme recommended

2. icon.png (in parent folder, 128x128 pixels)
   - Mod icon shown in mod list
   - Square format

Color Palette:
- Primary: #6930c3 (Purple)
- Secondary: #8B5CF6 (Light Purple)
- Background: #0f0c29 to #302b63 (Dark Blue to Purple gradient)
- Text: #FFFFFF (White)
- Success: #55FF55 (Green)
- Error: #FF5555 (Red)
- Warning: #FFFF55 (Yellow)

Note: If textures are missing, the mod will use fallback text rendering.
